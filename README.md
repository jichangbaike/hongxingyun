## 3. 协议与线路架构

### 3.1 Trojan 中转

- TLS 加密
- Web 标头伪装
- 更强抗封锁性
- 延迟稳定
- 日常通勤/办公更适合

相比 SSR / VMess：
- 连接成功率更高
- 协议生命周期更长
- 市面高质量机场普遍迁移至 Trojan

---

### 3.2 Hysteria2 直连

- UDP + BBR-like 自适应算法
- 超高吞吐效率
- 对丢包与延迟敏感环境极友好

适合：
- 4K 视频
- 大文件传输
- AI API / ChatGPT / Claude
- 晚高峰高速拉流

> **一句话总结**：Trojan 稳定保底，Hysteria2 性能上限。
