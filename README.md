## 四、协议与连接方式

红杏云提供 **VLESS、AnyTLS、Hysteria2** 等协议。不同协议的连接表现会受到本地网络、运营商、客户端及节点负载影响。

### 4.1 VLESS

VLESS 是目前较常见的轻量化协议之一，通常具有较好的客户端兼容性，适合日常网页访问、办公和常规视频场景。

### 4.2 AnyTLS

AnyTLS 采用 TLS 流量形态，主要用于改善连接稳定性和网络适应能力。实际效果与客户端版本及节点配置有关。

### 4.3 Hysteria2

Hysteria2 基于 UDP，通常在高延迟或存在一定丢包的网络环境中具有较好的吞吐表现，可用于高清视频、大文件传输及 AI 工具访问等场景。

> 实际使用时不必固定选择某一种协议。建议分别测试延迟、速度和稳定性，再选择更适合当前网络的节点。

## 十一、常见问题

### 11.1 红杏云支持多少台设备？

现有资料显示支持多设备使用，具体是否限制同时在线设备数，应以套餐页面和服务条款为准。

### 11.2 包月流量用完后怎么办？

可查看是否支持购买流量重置包，或者升级至更高流量套餐。

### 11.3 不限时套餐适合长期使用吗？

不限时套餐适合低频、备用或流量消耗不固定的用户，但一次性购买金额较高，不建议仅因单价较低就盲目购买大流量套餐。

### 11.4 为什么同一个节点在不同设备上的速度不同？

可能与客户端内核、协议支持、Wi-Fi 信号、本地运营商、设备性能及分流规则有关。

### 11.5 为什么流媒体有时可以观看，有时无法解锁？

流媒体平台会动态识别和限制数据中心 IP。节点的解锁状态可能随时间变化，可尝试切换同地区的其他节点。
